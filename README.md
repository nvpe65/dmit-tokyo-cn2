## CN2 GIA 是什么，为什么东京节点要特别看这个

**CN2 GIA（China Telecom Global Internet Access）** 是中国电信旗下的顶级骨干网线路，AS 号是 4809。和普通国际线路相比，它单独维护、不走拥堵的 163 骨干网，所以在国内用户大量上网的晚高峰，它的表现要稳定得多。

日本到中国大陆的物理距离比美西近不少，加上 CN2 GIA 加持，东京机房在延迟和稳定性上有天然优势——对比洛杉矶的 150ms 上下，东京到国内大概在 60-100ms 区间，差距很明显。

当然，CN2 GIA 不是万能的。如果你的用户在欧美，或者主要是东南亚普通流量，买它就是在为你用不到的东西付钱。
